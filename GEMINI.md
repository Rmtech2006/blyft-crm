# Gemini Agent Instructions

This project uses [Convex](https://convex.dev) as its backend.

When working on Convex code, **always read `convex/_generated/ai/guidelines.md` first** for important guidelines on how to correctly use Convex APIs and patterns.

## Superpowers Skills

This project uses the [Superpowers](https://github.com/obra/superpowers) methodology. Skills are in `docs/superpowers/skills/`. Read the relevant skill file before acting on any task.

**Mandatory — read before every task:**
- Creating features/components → `docs/superpowers/skills/brainstorming.md`
- Any bug or failure → `docs/superpowers/skills/systematic-debugging.md`
- Writing implementation code → `docs/superpowers/skills/test-driven-development.md`
- Claiming work is done → `docs/superpowers/skills/verification-before-completion.md`

**On-demand:**
- Multi-step tasks → `writing-plans.md`
- Running a plan → `executing-plans.md`
- Parallel independent tasks → `dispatching-parallel-agents.md`
- Isolated feature branch → `using-git-worktrees.md`
- Before merge → `requesting-code-review.md`
- Handling review feedback → `receiving-code-review.md`
- Implementation complete → `finishing-a-development-branch.md`

**Priority:** User instructions → Superpowers skills → Default behavior
